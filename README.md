# NLW18-React-Native

Create the project as Navigation (TypeScript):
```sh
npx create-expo-app@latest --template
```

Install the dependencies:
```sh
npx expo install expo-font @expo-google-fonts/rubik
```

Start the project:
```sh
npx expo start
```
